# Library-Management-System
The Library Management System is a Java-based console application designed to automate and simplify the management of library operations. The system allows librarians to efficiently manage books and perform common library tasks such as adding books, viewing available books, searching for books, issuing books to users, and returning issued books.

The project is developed using Core Java and follows an object-oriented programming approach. It provides a menu-driven interface through which users can interact with the system. The application maintains book details such as Book ID, Title, Author, and Availability Status.
